როგორ დავარენდეროთ JSX ელემენტები:

1. პირველად საჭიროა React და ReactDOM-ის იმპორტი:
   import React from 'react';
   import ReactDOM from 'react-dom/client';

2. შემდეგ შექმენი JSX ელემენტი:
   const element = <h1>Hello, world!</h1>;

3. უნდა მიუთითო სად უნდა იყოს ელემენტის აჩვენება HTML-ში:
   // HTML ფაილში უნდა იყოს div id="root"
   <div id="root"></div>

4. გამოყენე ReactDOM-ს ელემენტის რენდერისთვის:
   const root = ReactDOM.createRoot(document.getElementById('root'));
   root.render(element);
