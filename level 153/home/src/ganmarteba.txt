რა არის React:
React არის JavaScript ბიბლიოთეკა, რომელიც გამოიყენება ვებგვერდებზე მომხმარებლის ინტერფეისების (UI) შესაქმნელად.
ის საშუალებას გვაძლევს დავწეროთ კომპონენტური კოდი  ანუ დავყოთ გვერდი მცირე, ხელახალად გამოყენებად ნაწილებად
React ქმნის სწრაფ და რეაქტიულ ვებსაიტებს რადგან ის მხოლოდ იმ ნაწილს აახლებინებს რომელიც შეიცვალა


რისთვის გამოიყენება React:
React გამოიყენება დინამიკური ვებაპლიკაციების შესაქმნელად, სადაც მონაცემები ხშირად იცვლება.
მაგალითად სოციალური ქსელები ონლაინ მაღაზიები ჩატები და ინტერნეტ აპები
React-ის დახმარებით მარტივად იქმნება სწრაფი და თანამედროვე ინტერფეის რომელიც რეალურ დროში განახლდება მომხმარებლის მოქმედებისას


რა არის JSX:
JSX (JavaScript XML) არის სინტაქსი, რომელიც საშუალებას გვაძლევს JavaScript-ში HTML-ის მსგავსი კოდი დავწეროთ
JSX ამარტივებს React კომპონენტების შექმნას და კოდის წაკითხვას
ბრაუზერი პირდაპირ JSX-ს ვერ ამუშავებს, ამიტომ ის გადაკეთდება ჩვეულებრივ JavaScript კოდად, როცა აპლიკაცია იქმნება
